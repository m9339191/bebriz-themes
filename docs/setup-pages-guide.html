Setting Up Pages

How to add About Us, Contact, FAQ, and Blog pages to your theme

Applies to all beBriz themes

Brewmark, Pump, Matcha, Espresso, Caramel, Roastery, and all other themes include custom page templates for About Us, Contact, and FAQ. Follow these steps to activate them.

1. Create the About Us Page

Step 1: Add a new page

  1. Go to your Shopify Admin
  2. Navigate to Sales channels → Online Store → Pages
  3. Click "Add page"
  4. Title: About Us
  5. In the right sidebar, find "Theme template"
  6. Select page.about from the dropdown
  7. Set Visibility to "Visible"
  8. Click Save

Step 2: Add to the navigation menu

  1. Go to Online Store → Navigation
  2. Click on "Main menu"
  3. Click "Add menu item"
  4. Name: About Us
  5. Link: Click "Pages" → select "About Us"
  6. Click Save menu
Tip: You can customize the About Us content in the Theme Editor: Online Store → Themes → Customize → Pages → About Us. Edit headings, text, team members, and images directly.

2. Create the Contact Page

  1. Go to Sales channels → Online Store → Pages
  2. Click "Add page"
  3. Title: Contact
  4. Theme template: Select page.contact
  5. Set Visibility to "Visible"
  6. Click Save
  7. Add to Main menu (same steps as above)

3. Create the FAQ Page

  1. Go to Sales channels → Online Store → Pages
  2. Click "Add page"
  3. Title: FAQ
  4. Theme template: Select page.faq
  5. Set Visibility to "Visible"
  6. Click Save
  7. Add to Main menu or Footer menu

4. Set Up the Blog (Journal)

Step 1: Create a blog

  1. Go to Sales channels → Online Store → Blog posts
  2. Click "Add blog post"
  3. Write your first post (title, content, featured image)
  4. In the right sidebar under "Blog", select or create "Journal"
  5. Set Visibility to "Visible"
  6. Click Save

Step 2: Add to navigation

  1. Go to Online Store → Navigation → Main menu
  2. Click "Add menu item"
  3. Name: Journal (or Blog)
  4. Link: Click "Blogs" → select "Journal"
  5. Click Save menu

5. Enter Your License Key

  1. Go to Online Store → Themes → Customize
  2. Click "Theme settings" (gear icon at bottom left)
  3. Find the "License" section
  4. Paste your license key (from your purchase confirmation email)
  5. Click Save
  6. The "Theme by beBriz" watermark will be removed
